= GRB

  re-implement git_remote_branch [http://github.com/webmat/git_remote_branch] for cleaner,simpler,stronger!

== USAGE:
	grb pull [branch]  (default current_branch)
	grb mv   [branch1] [branch2]
	grb mv   [branch]  (default current_branch)
	grb push [branch]  (default current_branch)
	grb rm   [branch]  (default current_branch)
	grb new  [branch]
